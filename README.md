# UZK-simulation
UZK‑simulation – это полноценный инженерный симулятор установки замедленного коксования (УЗК) тяжёлых нефтяных остатков. Программа позволяет:
моделировать полный технологический цикл (нагрев сырья → коксование → ректификация → газофракционирование → глубокая переработка газа);
подбирать оптимальные параметры процесса (температура, давление, расход пара, рециркуляция);
оценивать тепловые и материальные балансы, экономическую эффективность и экологические показатели;
прогнозировать выходы продуктов с помощью нейронной сети (TensorFlow.js).
Проект предназначен для технологов НПЗ, инженеров‑проектировщиков, студентов нефтегазовых специальностей и всех, кто интересуется глубокой переработкой нефти.
# Клонируйте репозиторий
git clone https://github.com/fanya789/UZK-simulation.git
cd UZK-simulation

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev
После запуска откройте http://localhost:5173 в браузере.
# 🖥 Использование
Параметры – выберите сырьё, настройте температуру печи, давление, расход пара и т.д. Нажмите «Запустить моделирование» (или дождитесь автоматического пересчёта через 0,5 секунды после изменения параметров).

Схема – интерактивная P&ID‑схема УЗК с отображением текущих расходов продуктов.

Печь – подберите диаметр труб и расход пара для змеевика.

Камера – задайте геометрию коксовой камеры и получите тепловой баланс с расчётной температурой верха.

Газофракционирование – узнайте, сколько сухого газа, этан‑этилена, пропан‑пропилена, бутан‑бутилена и бензина получается из жирного газа.

Глубокая переработка – добавьте очистку, осушку и низкотемпературную деэтанизацию.

Нейросеть – обучите модель (кнопка «Обучить нейросеть») и затем получайте прогнозы мгновенно.

Экономика и Экология – оцените финансовую эффективность и воздействие на окружающую среду.

Экспорт CSV – сохраните все данные для отчёта.
